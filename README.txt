This fork is to run MNE in SCIRun (https://github.com/SCIInstitute/SCIRun).  
To use this package, download the `scirun_install` branch of this fork:

`git clone https://github.com/jessdtate/mne-python.git --single-branch --branch=scirun_install --depth=1`

navigate to the mne_python directory `cd mne_python` then build the package with a matching version of python with
`pip install .`

see (https://scirun.readthedocs.io/en/latest/user_doc/python.html#installing-packages) for more info
